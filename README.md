# Find The Pair Game

## Compilation Commands

Microsoft C/C++ Compiler 

``` 
cl /EHsc /Ox /W4 /Ferun.exe ftp.cl.cpp
```

GNU Compiler Collection 

``` 
g++ -Ofast ftp.gcc.cpp -o run.exe
```
